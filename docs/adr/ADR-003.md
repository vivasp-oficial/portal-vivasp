# ADR-003: Renderização Condicional de Cabeçalho e Rodapé via Elementor

## Contexto

O tema permite que o cabeçalho e rodapé sejam renderizados via Elementor se um post específico estiver configurado nas opções ou meta dados.

## Decisão

Verificar se o Elementor está ativo e se o post configurado existe, então usar `Empath_Plugin_Helper::empath_render_header/footer` para renderizar, caso contrário usar templates padrão.

## Alternativas consideradas

- Sempre usar templates PHP padrão.
- Forçar uso do Elementor para cabeçalho e rodapé.

## Trade-offs

- **Prós:** Flexibilidade para o usuário customizar via Elementor.
- **Contras:** Dependência do plugin Elementor, complexidade adicional no tema.

## Consequências

- Se Elementor não estiver ativo, fallback para templates padrão.
- Pode haver inconsistências visuais se o cabeçalho/rodapé Elementor estiver desatualizado.

Arquivo base: `inc/qubar-helper-class.php`.
