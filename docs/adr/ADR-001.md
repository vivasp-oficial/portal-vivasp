# ADR-001: Uso do Framework CS para Opções e Meta Boxes

## Contexto

O tema utiliza o framework CS para gerenciar opções do tema e meta boxes para posts e páginas, acessado via funções como `cs_get_option()` e `get_post_meta()`.

## Decisão

Adotar o framework CS para centralizar e simplificar o gerenciamento de opções e meta dados do tema.

## Alternativas consideradas

- Criar sistema próprio de opções e meta boxes.
- Usar outro framework ou API nativa do WordPress para opções.

## Trade-offs

- **Prós:** Facilita a criação e manutenção das opções, integração pronta com painel de administração.
- **Contras:** Dependência de um framework externo, possível dificuldade para migrar ou atualizar.

## Consequências

- O tema depende do framework CS estar ativo e configurado corretamente.
- Facilita a personalização do tema via painel.
- Pode dificultar a compatibilidade futura se o framework deixar de ser mantido.

Arquivo base: `inc/cs-framework-functions.php`, uso em `functions.php`.
